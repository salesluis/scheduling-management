# Implementar CRUD de Professional com Horários de Trabalho

## Descrição
Desenvolver um sistema para gerenciar os profissionais e seus horários de trabalho.

## Funcionalidades
- Criar, ler, atualizar e deletar (CRUD) informações de profissionais.
- Associar horários de trabalho aos profissionais.

## Requisitos
- Interface de usuário para gerenciar profissionais.
- Validação de entrada para os horários de trabalho.

## Entregas
- Código fonte do projeto.
- Documentação de uso e instalação.
